    RA1

    Actividades¶

    La unidad 1 tiene como objetivo principal adquirir el RA1, el cual se define como:

    Resultado de aprendizaje 1 de Sistemas Operativos en Red

    • RASOR.1 Instala sistemas operativos en red describiendo sus características e interpretando la documentación técnica.
      • CE1a: Se ha realizado el estudio de compatibilidad del sistema informático.
      • CE1b: Se han diferenciado los modos de instalación.
      • CE1c: Se ha planificado y realizado el particionado del disco del servidor.
      • CE1d: Se han seleccionado y aplicado los sistemas de archivos.
      • CE1e: Se han seleccionado los componentes a instalar.
      • CE1f: Se han aplicado procedimientos para la automatización de instalaciones.
      • CE1g: Se han aplicado preferencias en la configuración del entorno personal.
      • CE1h: Se ha actualizado el sistema operativo en red.
      • CE1i: Se ha comprobado la conectividad del servidor con los equipos cliente.

    A continuación se van a detallar las actividades de enseñanza aprendizaje oportunas para dicho cometido.

    Práctica 1_1 Instalación de Sistemas Operativos en Red¶

    Siguiendo la situación de aprendizaje definida al Inicio del módulo, estamos trabajando en el departamento TIC de NetOS y nos encargan instalar los últimos SOs de Windows y Ubuntu en máquinas virtuales sobre el hipervisor VirtualBox.

    Los sistemas opertivos se deben instalar tanto en la versión servidor como en la versión cliente, con el objetivo de preparar un escenario virtual de Sistemas Operativos en Red para pruebas de servicio de directorio, a instalar y configurar más adelante.

    • Sistemas Operativos a instalar:
    SOs Descarga
    Windows Server 2022 Datacenter (updated July 2023, Español) Azure Portal
    Windows 11 Education N, Version 23H2 (Español) Azure Portal
    Ubuntu Server 24.04.1 LTS Ubuntu Server
    Ubuntu Desktop 24.04.1 LTS Ubuntu Client

    Tu coordindor TIC te ha pedido que generes un informe (Máximo 30 páginas), se debe cuidar bien el formato, con los siguientes pasos:

    1. Realiza un estudio de compatibilidad de los SOs a instalar con el sistema informático anfitrión.

      • En el informe aporta simplemente el resumen de las conclusiones principales del estudio. (CE1a)

    Consejo

    Antes de crear las MVs en cualquier sistema informático, se ha de tener en cuenta los requisitos con los que dispone el sistema anfitrión y que, en circunstancias normales, serán necesarias dos de estas máquinas encendidas a la vez. Debes planificar bien el consumo de recursos. esta información debes obtenerla a través de la navegación de las páginas oficiales de los SOs, donde se especifiquen los requisitos mínimos.

    1. Descarga las imágenes de los sistemas operativos, crea e instala una máquina virtual en Virtualbox. En el informe debes adjuntar las capturas de la instalación de cada SO que respondan a las siguientes cuestiones:

      • Modo de instalación. (CE1b)
      • Selección del disco. (CE1c)
      • Selección del sistema de archivos. (CE1d)
      • Componentes a instalar. (CE1e)
      • Preferencias del entorno personal. (CE1g)
      • Actualizaciones y automatizaciones. (CE1f y CE1h)

    Indicaciones a seguir

    • Deberás usar todo el disco duro para la instalación.
    • El nombre de cada máquina queda a tu discreción, aunque se recomiendan nombres cortos significatibos de cada distribución enunciada.
    • Instala las Guest Additions en cada máquina, esto facilitará el trabajo con ellas (En ubuntu Server no es necesario).

    1. Crea una red virtual interna NAT para cada servidor e introduce en ella todas las máquinas virtuales separadas por sistemas operativos; los de Microsoft en una red y los de GNU/Linux en otra. Asegúrate que las direcciones MAC de las tarjetas de red no se repitan.

      • En el informe debes adjuntar las capturas de las pruebas de conexión (Ping), entre servidor y cliente de cada distribución. (CE1i)

    2. Realiza una instántanea inicial de cad MV, para guardar esta primera instalación y configuración, y poder volver a ella cuando sea necesario.
